Study of Principle Component Analysis

The objective of this project was to extract the principal components of a given dataset and visualize them in two ways:

  • Implementing Principal Component Analysis (PCA) from scratch.
  • Using the pre-built PCA class from sklearn.decomposition.

After performing PCA through both methods, I compared the results of dimensionality reduction with the original dataset. To assess the impact of dimensionality reduction on model performance, I applied a classification algorithm (K-Nearest Neighbors in this case) and measured the accuracy of the model with and without PCA.
